Basic Computer Terms

Algorithm

A set of instructions designed to perform a specific task. More about algorithms.

Application (App)

A program designed to perform a group of coordinated functions, tasks, or activities for the benefit of the user.

Bit

The smallest unit of data in computing represented by a 0 or 1.

Byte

A group of 8 bits. Learn about bytes.

CPU (Central Processing Unit)

The primary component of a computer that performs most of the processing inside a PC. CPU explained.

Cloud Computing

The practice of using a network of remote servers hosted on the Internet to store, manage, and process data.

Data

Raw, unorganized facts that need to be processed.

Database

An organized collection of data, generally stored and accessed electronically from a computer system.

Hardware

The physical parts of a computer system. Understanding computer hardware.

Internet

A vast network that connects computers all over the world. About the Internet.

Memory

The component of a computer that stores data.

Operating System (OS)

The software that supports a computer's basic functions. Operating systems overview.

Peripheral

External devices connected to the computer.

Program

A set of instructions that a computer uses to perform a specific function.

Server

A system that provides resources to other computers over a network.

Software

A collection of instructions that enables interaction with a computer.

URL (Uniform Resource Locator)

The address of a web page on the world wide web. URLs explained.

Virus

A malicious program that can corrupt the system or destroy data.

Wi-Fi

A technology that allows computers to communicate over a wireless signal.

User Interface (UI)

The means by which a user and a computer system interact. Exploring User Interfaces.
